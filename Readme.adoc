# 🚗 Aplicación de Gestión de Vehículos y Reservas

Aplicación móvil desarrollada para gestionar la consulta de vehículos y la creación de reservas de forma sencilla mediante una aplicación Android conectada a un backend.

La aplicación permite a los usuarios registrarse, iniciar sesión, consultar vehículos disponibles y gestionar sus reservas directamente desde el dispositivo móvil.

# 📱 Funcionalidades principales

## Cliente registrado

- Inicio de sesión de usuario
- Recuperación de contraseña
- Edición del perfil de usuario
- Cierre de sesión
- Crear una reserva
- Ver las reservas propias
- Ordenar reservas por fecha de inicio
- Ver el detalle de una reserva
- Anular una reserva antes de que comience

## Cliente anónimo

- Registro de usuario
- Listar vehículos disponibles
- Ver el detalle de un vehículo
- Ordenar vehículos por precio
- Consultar disponibilidad de vehículos por rango de fechas

# 🏗 Arquitectura del sistema

El proyecto está formado por tres partes principales:

- **Aplicación Android** que actúa como cliente.
- **Backend desarrollado con Spring Boot** que expone una API REST.
- **Base de datos MySQL** para almacenar la información del sistema.

La aplicación Android se comunica con el backend mediante peticiones HTTP a la API REST.
